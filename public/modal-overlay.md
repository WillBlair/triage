# Version 2 — Conversation History Modal Overlay

## Overview
When a doctor clicks "View conversation (N msgs)" in the Follow Up
detail panel, a modal overlay opens containing an AI-generated clinical
summary at the top and the full chat transcript below it.

---

## Trigger
- Button label: "View conversation (N msgs)"
  - N = total message count from conversation_messages table
  - Example: "View conversation (7 msgs)"
- Button style: teal outline (#1D9E75 border, transparent background)
- Located in the Follow Up detail panel below symptoms and patient note

---

## Behavior & Accessibility

### Opening
- Triggered by "View conversation (N msgs)" button click
- Background dims: rgba(0,0,0,0.35) overlay
- Modal animates in (fade or scale, optional)
- Focus trapped inside modal while open

### Closing
- Click the × button in the header
- Click the "Close" button in the footer
- Press Escape key
- Click the dimmed backdrop outside the modal

### Scroll
- Modal body scrolls independently
- Modal itself does not scroll with the page
- Max height approximately 80vh

## Supabase Tables Required

### conversation_messages
```sql
create table if not exists conversation_messages (
  id uuid primary key default gen_random_uuid(),
  checkin_response_id uuid references checkin_responses(id),
  sender text not null check (sender in ('bot', 'patient')),
  message text not null,
  sent_at timestamptz default now()
);

alter publication supabase_realtime
  add table conversation_messages;
```

### checkin_responses — add summary cache column
```sql
alter table checkin_responses
  add column if not exists conversation_summary text;
```

## AI Summary Generation

### API call
```javascript
const response = await fetch("https://api.anthropic.com/v1/messages", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    model: "claude-sonnet-4-20250514",
    max_tokens: 1000,
    messages: [
      {
        role: "user",
        content: `Summarize this patient check-in conversation in 
        2-3 sentences from a clinical perspective. Focus on symptoms 
        reported, severity, and any flags triggered.
        
        Conversation:
        ${messages.map(m => `${m.sender}: ${m.message}`).join('\n')}`
      }
    ]
  })
});
```

### Caching logic
```
1. On modal open, check checkin_responses.conversation_summary
2. If not null → display cached summary immediately
3. If null → call Anthropic API → display result → save to
   checkin_responses.conversation_summary via Supabase update
4. Never regenerate if cache exists
```
